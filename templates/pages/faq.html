Frequently Asked Questions

What is Musicakes?

Musicakes is a simple, experimental web application for artists to sell their music, and to tokenise the revenue streams from their music sold on this platform.

To purchase music, a user will have to pay using the DAI token, a USD-pegged stablecoin, to a smart contract specific to that release.

We enable the artist to determine how the payment should be split by representing shares to the proceeds as tokens (called Musicakes too). Payments to the release's smart contract will be allocated based on how the Musicakes for that release is distributed across wallet addresses. Each release will have 100 Musicakes tokens upon the creation of the smart contract. All 100 tokens will be sent to the artist's wallet address that was used at the time the release was created. We may implement a 2% fee in the future to cover operating costs for AWS storage.

In order to claim payment, a user will need to trigger a function in the smart contract using the Ethereum wallet address holding the Musicakes token. This will require payment of a gas fee to the Ethereum network.

What do I need to use Musicakes?

To use Musicakes, you need to install Metamask and an Ethereum wallet address with DAI tokens.

You can purchase DAI using ETH at Uniswap, a decentralised exchange on the Ethereum blockchain.

What is DAI token?

DAI is a token on the Ethereum blockchain that is pegged to the value of one USD per token.

Why is my purchase not showing up?

The transaction on the Ethereum network may take a while to be confirmed. You should not leave the release page until the transaction is confirmed. If you do, please send us your transaction hash and email account at musicakes [dot] team [at] gmail [dot] com so that we can update the database.

Can I use different wallet addresses?

Yes, you can. Your account is not tied to a wallet address.

Do you use decentralised file storage?

No. We are currently using AWS S3 to store the track files.

Do you have an email that I can reach out to?

You can address your queries to musicakes [dot] team [at] gmail [dot] com.
